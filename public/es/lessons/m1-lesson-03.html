Condiciones: si es creeper, entonces corre

Cómo enseñar un programa a tomar decisiones.

🎣 Empezar

Estás caminando por una cueva en Minecraft. Hay una multitud adelante. ¿Qué estás haciendo?

  • si esto vaca → ignorar
  • si esto zombi → golpeas con una espada
  • si esto enredadera → CORRE 💨
  • si esto Enderman → no lo mires a los ojos

Acabas de cumplir cuatro condiciones. Tu cerebro hace esto automáticamente: evalúa la situación y elige una acción. Los programas funcionan de la misma manera, pero deben especificar cada condición explícitamente.

🧠 Teoría: si/si no

Condición - Esta es una bifurcación en el código. Si algo es verdad, hacemos una cosa. Si no, algo más.

multitud = "enredadera"

if multitud == "enredadera":
    print("¡VAMOS A CORRER!")
else:
    print("¡Ataquemos!")

Analicémoslo pieza por pieza:

  • if - "si"
  • mafia == "enredadera" — comprobar (dos caracteres = - esta es una comparación, una = - asignación)
  • : - dos puntos después de la condición (Python lo quiere así, no discutas)
  • Sangría (4 espacios): qué se hace si la condición es verdadera
  • else: - “de lo contrario” (todo lo demás)

elif - cuando hay más de dos opciones

multitud = "zombi"

if multitud == "enredadera":
    print("¡VAMOS A CORRER!")
elif multitud == "zombi":
    print("Golpeamos con una espada")
elif multitud == "esqueleto":
    print("Escóndete detrás del escudo")
elif multitud == "enderman":
    print("No hagas contacto visual")
else:
    print("Algo desconocido. Ten cuidado".)

elif es una abreviatura de "si no". Puedes apostar tanto como quieras.

Tarea 1: clasificador de mafia

En campos vacíos ??? introduce los comandos: if (si) elif (y si) o else (en caso contrario). Estas son las palabras que le indican al programa cuándo ejecutar el código.

multitud = "lobo"

 multitud == "enredadera":
    tipo = "hostil"
 multitud == "lobo":
    tipo = "neutro"
:
    tipo = "amigable"
💡 Pista

La primera comprobación es si. Verificación adicional - elif. Todo lo demás es otra cosa.

✅ Solución 
if multitud == "enredadera":
    tipo = "hostil"
elif multitud == "lobo":
    tipo = "neutro"
else:
    tipo = "amigable"
