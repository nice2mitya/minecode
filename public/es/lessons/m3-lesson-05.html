La recursividad es una función que se llama a sí misma.

Resolvemos el problema a través de una copia más pequeña de nosotros mismos.

🎣 Hook

La rama del árbol es como un árbol pequeño. Muchas tareas se estructuran de la misma manera: “grande = pequeño + paso”.

🧠 Concept

Términos: lista, diccionario, bucle, función: escribimos en inglés, explicamos en ruso.

La recursividad funciona cuando hay:
1) caso base (cuándo detenerse);
2) paso recursivo (cómo reducir el problema).

Ejemplo factorial:
hecho(n) = n * hecho(n-1), si n>1; de lo contrario 1.

🔧 Practica

Tarea 1: completar el caso base n == 1.

if n == 1:
    return 1

Complete los espacios en blanco del código en su cuaderno/editor. Respuestas clave: 1.

Tarea 2: agregar una llamada recursiva para la suma 1...n.

return n + sum_to_n(n-1)

Ponte a prueba: n-1.
