Funciones: crea tus propios equipos

Escribí una vez, llama tantas veces como quieras.

🎣 Empezar

Ya sabes cómo construir un muro. ¿Qué pasa si necesitas 10 paredes diferentes en diferentes lugares? ¿Copiar el código 10 veces?

No. Creas una función: tu propio equipo. Lo escribí una vez, llama tantas veces como quieras.

Una función es como una receta de elaboración. Una vez que escribas la receta del pico, podrás crear tantos picos como quieras sin tener que recordar la ubicación de los ingredientes cada vez.

🧠 Teoría: Funciones

def nombre(): - crear una función

palabra clave def (de “definir” - definir) crea un nuevo comando:

def construir_una casa():
    levantar_paredes()
    poner_el_techo()
    poner_la_puerta()

Ahora construir_casa() - tu equipo personal. Escribí una vez: se construyen casas en todas partes:

construir_una casa()   # casa numero 1
adelante()
construir_una casa()   # casa numero 2
adelante()
construir_una casa()   # casa numero 3

Configuración - Funciones personalizables

¿Quieres paredes de diferentes tamaños? Agregar parámetros:

def construir_una pared(ancho, alto):
    for cuerda in range(altura):
        for bloquear in range(ancho):
            poner_bloque()

construir_una pared(10, 5)   #pared 10×5
construir_una pared(3, 20)    # torre 3×20

Los parámetros son como los ingredientes de una receta. Diferentes ingredientes → diferentes resultados.

retorno - resultado de la función

Una función no sólo puede hacer algo, sino también resultado de retorno:

def zona(a, b):
    return a*b

resultado = zona(10, 5)   # resultado = 50

Analogía: función = receta de elaboración. Parámetros = ingredientes. return = resultado de elaboración.

💭 Resultados

Ahora lo sabes:

  • ✅ def - crea una nueva función
  • ✅ Parámetros: haga que la función sea personalizable
  • ✅ return - devuelve el resultado
  • ✅ Función = escrita una vez, usada en todas partes

Funciones en el mundo real:

  • 📱 Cada botón de la aplicación = llamada de función
  • 🖨️ print() - también una función (recién escrita por alguien antes que tú)
  • ⛏️ Minecraft completo = miles de funciones que se llaman entre sí
