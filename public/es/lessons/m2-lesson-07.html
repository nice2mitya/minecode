👹 Nivel de jefe: sistema de inventario

Proyecto final del módulo: arquitectura de inventario con búsqueda, filtros y eventos.

🎣 Hook

Construyes un sistema de inventario completo, como en un juego real: estructura de datos + eventos + estados + modularidad.

🧠 Concept

Términos: lista, diccionario, bucle, función: escribimos en inglés, explicamos en ruso.

El proyecto combina todo desde el módulo:
- matrices y diccionarios para almacenar elementos;
- estructuras anidadas para categorías;
- Observador para actualizar la interfaz de usuario;
- Estado de los modos de inventario;
- modularidad para escalar.

🧩 Prueba +10 XP

🔧 Practica

Etapa 1: complete las claves del artículo: identificación y rareza.

item = {"id": "iron_sword", "rarity": "rare"}
print(item)

Complete los espacios en blanco del código en su cuaderno/editor. Respuestas clave: id/rareza.

Etapa 2: agregar modos de estado de inventario.

state = "browse"
if click_item:
    state = "equip"

Ponte a prueba: navega/equipa.
