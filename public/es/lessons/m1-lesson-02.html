Variables - Inventario de datos

Donde el programa almacena sus cosas.

🎣 Empezar

Tienes un inventario en Minecraft. 36 espacios. Puedes poner una cosa en cada uno: una espada, comida, bloques.

Ahora imagina: estás construyendo una casa. Debe recordar sus dimensiones: ancho 10, largo 15, alto 5. ¿Dónde guardar estos números?

Hay una manera de hacer esto en programación. variables. Son como espacios de inventario, pero para datos. Pones un valor allí, le das un nombre y lo usas cuando sea necesario.

🧠 Teoría: variables

variable es una ubicación de memoria con nombre. Ella tiene:

  • Nombre - como una firma en un cofre
  • Significado - lo que hay dentro
  • Tipo - qué tipo de datos
# Crear variables
ancho = 10
longitud = 15
altura = 5
materiales = "piedra"
hay un_techo = True
NombreSignificadoTipoAnalogía de Minecraft
ancho10número (int)Pila de 10 bloques
longitud15número (int)Pila de 15 bloques
altura5número (int)Pila de 5 bloques
materiales"piedra"texto (cadena)Firmar con inscripción
hay un_techoTruebooleano (booleano)Palanca: encendido/apagado

Tipos de datos básicos

  • int (entero) - 64, 0, -5. La cantidad de elementos en la pila.
  • float (número fraccionario) — 3.14, 0.5. Nivel de salud: 18,5 HP.
  • string (texto) — "Hola", "diamond_sword". Nombre del artículo.
  • bool (lógico) - True o False. Palanca: encendido o apagado.

Las variables se pueden cambiar

Esto es lo mismo reintercambiable. El valor se puede actualizar:

salud = 20        # HP completo
salud = 20 - 7    # Una enredadera explotó cerca
salud = 13        # Ahora 13 CV

Y úselo en los cálculos:

ancho = 10
longitud = 15
zona = ancho * largo    # 150
perímetro = 2 * (ancho + largo)  #50 bloques de pared

¿Por qué es esto genial? Porque si decides cambiar el tamaño de la casa, necesitas cambiar UNO número, en lugar de reescribir todo el programa.

💡 Pista

El área de un rectángulo es la longitud por la altura. ¡Usa nombres de variables!

✅ Solución 
bloques_necesarios = longitud_pared * altura_pared    # 40

Tarea 3: Intercambio de valores

Intercambie el contenido de las dos ranuras. Pista: se necesita una tercera variable.

ranura_1 = "espada"
ranura_2 = "escudo"

# Intercambio:
temporal = 
ranura_1 = 
ranura_2 =
💡 Pista

Como dos cofres: primero coloque el primer artículo en el cofre temporal, luego mueva el segundo al primero, luego del temporal al segundo.

✅ Solución 
temporal = ranura_1     # temporal = "espada"
ranura_1 = ranura_2        # slot_1 = "escudo"
ranura_2 = temporal     # slot_2 = "espada"

  • ✅ Cuatro tipos básicos: números, texto, lógico
