Vibe coding — итеративная разработка с AI

Делаем проект короткими итерациями: запрос → проверка → доработка.

🎣 Hook

Vibe coding — не «нажал кнопку и готово». Это быстрые циклы уточнений, как стройка по чертежу с проверкой каждого этажа.

🧠 Concept

Термины: list, dictionary, loop, function — пишем по-английски, объясняем по-русски.

Итерация vibe coding:
1) даёшь ограниченную задачу;
2) получаешь код;
3) запускаешь/тестируешь;
4) даёшь точечный фидбек;
5) повторяешь.

Большие задачи дробим на мелкие.

🧩 Квиз +10 XP

A Огромные запросы на 500 строк
B Короткие проверяемые итерации
C Отказ от тестов

🔧 Практика

Задание 1: заполни шаги цикла: запрос → проверка → правка.

loop = ["запрос", "проверка", "правка"]

Заполни пропуски в коде в своей тетради/редакторе. Ключевые ответы: запрос/проверка.

Задание 2: укажи важный принцип итераций.

rule = "маленький шаг"

Проверь себя: маленький шаг.
