Prompt engineering — говори с AI точно

Учимся ставить задачу AI так, чтобы ответ был проверяемым и полезным.

🎣 Hook

Команда /give в Minecraft работает, только если параметры точные. С AI так же: неясный запрос = неясный результат.

🧠 Concept

Термины: list, dictionary, loop, function — пишем по-английски, объясняем по-русски.

Хороший промпт включает:
- роль/контекст;
- конкретную цель;
- входные данные;
- формат ответа;
- критерии проверки.

Не «сделай красиво», а «дай 3 варианта API с плюсами/минусами».

🔧 Практика

Задание 1: добавь обязательный блок «формат ответа».

prompt = "... формат: table из 3 пунктов"

Заполни пропуски в коде в своей тетради/редакторе. Ключевые ответы: формат.

Задание 2: укажи требование к верификации.

prompt += " + проверка граничных случаев"

Проверь себя: проверка.
