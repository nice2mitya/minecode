Recursion is a function that calls itself

We solve the problem through a smaller copy of ourselves.

🎣 Hook

The tree branch is like a small tree. Many tasks are structured the same way: “big = small + step”.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

Recursion works when there is:
1) base case (when to stop);
2) recursive step (how to reduce the problem).

Factorial example:
fact(n) = n * fact(n-1), if n>1; otherwise 1.
