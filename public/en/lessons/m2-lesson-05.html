State Pattern

One essence, different modes of behavior.

🎣 Hook

A player in Survival mode and in Creative mode are like two different characters. But this is one object with a different state.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

The State pattern allows you to switch the behavior of an object through the current state.

Example game states:
- lobby
- playing
- paused
- game_over

Transitions are as important as the states themselves: what can happen and in what order.
