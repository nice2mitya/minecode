Variables - Data Inventory

Where the program stores its things.

🎣 Start

You have an inventory in Minecraft. 36 slots. You can put one thing in each: a sword, food, blocks.

Now imagine: you are building a house. You need to remember its dimensions - width 10, length 15, height 5. Where to store these numbers?

There is a way to do this in programming. variables. These are like inventory slots, but for data. You put a value there, give it a name and use it when needed.

🧠 Theory: Variables

Variable is a named memory location. She has:

  • Name - like a signature on a chest
  • Meaning - what lies inside
  • Type - what type of data
# Create variables
width = 10
length = 15
height = 5
material = "stone"
there is a_roof = True

Basic Data Types

  • int (integer) — 64, 0, -5. The number of items in the stack.
  • float (fractional number) — 3.14, 0.5. Health level: 18.5 HP.
  • string (text) — "Hello", "diamond_sword". Item name.
  • bool (logical) - True or False. Lever: on or off.

Variables can be changed

This is the same reexchangeable. The value can be updated:

health = 20        # Full HP
health = 20 - 7    # A creeper exploded nearby
health = 13        # Now 13 HP

And use it in calculations:

width = 10
length = 15
area = width * length    # 150
perimeter = 2 * (width + length)  #50 wall blocks

Why is this cool? Because if you decide to change the size of the house, you need to change ONE number, rather than rewriting the entire program.

Task 1: Resource Calculator

You are building a rectangular wall. Fill out the formula to count blocks.

# Wall block calculator
wall_length = 10
wall_height = 4
blocks_needed =  *     # = 40
💡 Hint

The area of a rectangle is the length times the height. Use variable names!

✅ Solution 
blocks_needed = wall_length * wall_height    # 40

Task 3: Exchange of values

Swap the contents of the two slots. Hint: a third variable is needed.

# Exchange:

slot_1 =

💡 Hint

Like two chests: first put the first item in the temporary chest, then move the second to the first, then from the temporary to the second.

✅ Solution

slot_1 = slot_2        # slot_1 = "shield"
slot_2 = temporary     # slot_2 = "sword"

🏗️ Build: Customizable House

The turtle builds a house, the dimensions of which you set through variables.

# Home Settings
width = 6
length = 8
height = 4
material = "oak"

# Turtle builds according to your parameters
build_a house(width, length, height, material)

Change the variable values and run again. See how the same code builds different houses? This is the power of variables - you write the code once, and change the result by substituting other data.

💭 Results

Now you know:

  • ✅ Variable = named cell for data
  • ✅ Four basic types: numbers, text, logical
  • ✅ Variables can be changed and used in calculations
  • ✅ One program + different variables = different results

Easter egg: Do you know why a variable for exchanging values is called temp? This is short for temporary. Programmers are lazy - they are too lazy to write complete words. Get used to it. 😎
