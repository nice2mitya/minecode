Pathfinding - Mob AI

We find a path in the grid: from the start to the goal, avoiding obstacles.

🎣 Hook

The zombie knows how to reach the player, even if there are walls and corridors. This means he is looking for a path on the map.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

Pathfinding is often modeled as a graph:
- cells/points = vertices;
- transitions between them = edges.

For the “minimum number of steps” on equal edges, BFS (breadth first search) is suitable.
