Nested structures - a world of chunks

Building complex data: a list of dictionaries and a dictionary of lists.

🎣 Hook

One Minecraft chunk contains a bunch of blocks. One world - a bunch of chunks. Complex systems are made up of nested systems.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

Nested structures are when there are dictionaries inside a list, and lists inside a dictionary.

Example:
chunk = {
"coord": [0, 0],
"resources": ["coal", "iron"],
"mobs": [{"type": "zombies", "hp": 20}]
}

It is convenient to transfer such structures as JSON and store the state of the world.

Task 2: indicate the hp field key for the first mob.

chunk = {"mobs": [{"type": "zombie", "hp": 20}]}
