Reading AI code - like a reviewer

We check the AI code for logic, safety and edge cases.

🎣 Hook

AI can write working code, but miss one detail and the whole thing breaks. Your superpower: review and verification.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

Review checklist:
1) does the code solve the problem?
2) edge cases processed?
3) readability and names?
4) are there any dangerous places?
5) are there tests?

AI speeds up the draft, but the final quality is your responsibility.

🔧 Practice
