Dictionaries - book of enchantments

We retrieve data by key, not by position.

🎣 Hook

In the book of enchantments you are not looking for the “fifth line”, but for a specific name: “Witness”, “Mending”. This is the key-value.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

A dictionary (dict) stores pairs: key → value.

Example:
enchantments = {"sharpness": 3, "repair": 1}
print(enchantments["sharpness"])

When is a dictionary better than an array:
- need quick access by name;
- order is not as important as clear keys;
- the data is similar to an object card.
