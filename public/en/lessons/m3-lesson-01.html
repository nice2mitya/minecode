Linear search - looking for diamonds

Basic search: we check elements one by one.

🎣 Hook

Tunnel after tunnel you check each block until you find a diamond. This is a linear search.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

Linear search traverses the collection from left to right.

Pseudocode:
for element in data:
if element == target: return index

Pros: simplicity.
Cons: in the worst case, we check everything (O(n)).
