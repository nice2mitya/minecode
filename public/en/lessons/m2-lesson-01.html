Arrays - chest with compartments

How to store many values in one place and get what you need using an index.

🎣 Hook

Imagine: there are 27 slots in a chest. You don't call them "top left second right." You use the slot number. Arrays—collections with indexes—work in the same way in the code.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

An array is an ordered collection of elements. Each element has an index: 0, 1, 2...
In Python, an array is most often represented by a list.

Example:
blocks = ["earth", "stone", "diamond"]
print(blocks[0]) # ground
print(blocks[2]) # diamond

What's important:
- index starts from 0;
- you can change an element by index;
- it’s convenient to iterate through a for loop.
