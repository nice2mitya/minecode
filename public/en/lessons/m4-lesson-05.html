When AI lies - critical thinking

We recognize hallucinations and build a verification process.

🎣 Hook

AI can confidently tell lies. The danger is not in the mistake, but in the confident tone.

🧠 Concept

Terms: list, dictionary, loop, function - we write in English, we explain in Russian.

Typical risks:
- fictitious APIs/facts;
- incorrect links;
- skipping task constraints.

Antidote:
- checking sources;
- run the code;
- we check with the documentation;
- ask clarifying questions.

🧩 Quiz +10 XP
What is an AI hallucination?
A Beautiful response design
B Correct answer without source
C Convincing-sounding but false information
D Any long answer

🔧 Practice

action = "check the documentation"
